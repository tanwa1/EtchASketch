
# EtchASketch

An interactive web-based Etch A Sketch app built with HTML, CSS, and JavaScript.

## Features
- Create a customizable grid of squares.
- Draw by moving your mouse over the grid (squares turn black on hover).
- Responsive and modern UI.

## How It Works
1. Open the app in your browser (see below for how to run).
2. Click the "Click this!" button.
3. Enter the number of squares per side for your grid (e.g., 16 for a 16x16 grid).
4. Move your mouse over the grid to draw.

## Files
- `index.html`: Main HTML file.
- `etchASketch.js`: JavaScript logic for grid creation and drawing.
- `etchASketch.css`: Styles for layout and appearance.

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.

## Customization
- Change grid size by entering a different number when prompted.
- Modify styles in `etchASketch.css` for a different look.
